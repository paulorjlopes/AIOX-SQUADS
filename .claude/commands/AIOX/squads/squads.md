# AIOX Squads — Index

Lista completa de squads disponíveis do repositório xquads-squads.

## Como usar

Ative um squad com o comando `/AIOX:squads:{squad-name}` ou `@squad-name`.

**Exemplo:** `/AIOX:squads:c-level-squad` ativa o C-Level Squad com o Vision Chief.
