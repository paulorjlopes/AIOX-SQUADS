```

```

Then use commands:
- `*dissect https://example.com` — Analyze a website
- `*design` — Start a design project
- `*search https://example.com` — Search visibility audit

## 8 Dimensions of Web Excellence

1. **Structure & Patterns** — How the site is built
2. **Visual Design** — How it looks and feels (unique, not generic)
3. **Animation & Motion** — How it moves (purposeful, accessible)
4. **Design Tokens** — How design decisions are systematized
5. **Performance** — How fast it loads (Core Web Vitals)
6. **Accessibility** — How inclusive it is (WCAG 2.2)
7. **SEO** — How visible in search engines (Google, Bing)
8. **GEO** — How visible in AI search (ChatGPT, Perplexity, Gemini)
